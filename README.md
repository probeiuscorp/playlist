# Play**journey**

Tired of playlists being, well, lists? Playjournery allows you to create playlists that can randomly branch into different sequences, sprinkle in certain songs at random and ease the creation of large playlists by using the same sequence multiple times. Music can be added from YouTube or a local file.

Playjourney also supports adapting an existing YouTube playlist into its own format, making it even faster to transition.

## Technical

Playjourney is built with NodeJS, TypeScript, React, SASS and Webpack.
